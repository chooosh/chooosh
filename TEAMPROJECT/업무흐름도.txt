업무 흐름도 

전체적인 흐름 

회원 등록 -> 로그인 -> 프로젝트 모집 글 작성 혹은 참여 -> 매칭 -> 정보 관리


1. 회원가입 
	회원 유무 체크 (기존회원일 경우 로그인 폼으로 이동) 
	-> 신규회원 확인 후 회원정보 입력 폼으로 이동
	-> 회원가입데이터가 정상적으로 입력되었는지 확인(아이디 중복체크, 아이디/비번 자릿수 제한, 필수입력데이터 작성여부 등) 
	-> 데이터가 정상적으로 입력되었을 경우 회원가입 성공 메시지 팝업 후 로그인 폼으로 이동


2. 프로젝트 팀원 찾기 게시판 글 작성
	글 작성 버튼 클릭
	-> 로그인 유무 체크 (로그인 안되어있을경우 로그인 폼으로 이동)
	-> 글 작성 페이지로 이동
	-> 데이터 입력
	-> 데이터가 정상적으로 입력되었는지 확인 (필수데이터 입력, 글자 수 제한 등)
	-> 데이터가 정상적으로 입력되었을 경우 글 작성 완료	
	-> 글 목록 조회 페이지로 이동 

// 	수정/삭제버튼 클릭  
	(자기한테만 수정/삭제 버튼이 보이게할건지
	아니면 버튼 눌렀을때 비밀번호 누르게할건지?) 


3. 프로젝트 팀원 모집 게시판 글 작성
	글 작성 버튼 클릭
	-> 로그인 유무 체크 (로그인 안되어있을경우 로그인 폼으로 이동)
	-> 글 작성 기능로 이동
	-> 데이터 입력
	-> 데이터가 정상적으로 입력되었는지 확인 (필수데이터 입력, 글자 수 제한, 유료/무료모집 선택 등)
	-> 데이터가 정상적으로 입력되었을 경우 글 작성 완료	
	-> 글 목록 조회 페이지로 이동 

4. 프로젝트 팀원 모집 게시판 참여하기 기능
	참여하기 버튼 클릭
	-> 로그인 유무 체크(로그인 안되어있을 경우 로그인폼으로 이동)
	-> PR 메시지 작성 페이지로 이동 (혹은 팝업창?) 
	-> 글 작성 후 작성완료 버튼 클릭
	-> 유료모집일 경우 결제 api 페이지로 이동
	-> 글 작성 혹은 결제 완료 후 신청완료 안내 메시지 팝업 
	-> 원 모집글로 이동

// 	마이페이지에 신청현황 정보 업데이트 

5. 공지사항 게시판 글 작성
	글 작성 버튼 클릭	
	-> 로그인 유무 체크 (로그인 안되어있을 경우 로그인폼으로 이동)
	-> 관리자 여부 체크 (관리자 아닐 경우 팝업메시지)
	-> 데이터가 정상적으로 입력되었는지 확인 (필수데이터 입력, 글자 수 제한 등)
	-> 데이터가 정상적으로 입력되었을 경우 글 작성 완료	
	-> 글 목록 조회 페이지로 이동 

6. 투표 기능 (메인페이지)
	
7. 마이페이지 게시판 프로필 사진 변경
	로그인 여부 확인
	-> 회원 검증
	-> 프로필 사진 변경 버튼 클릭
	-> 이미지 변경 팝업 창 업로드
	-> 변경 후 변경완료 버튼 클릭
	-> 창 닫기
 
8. 마이페이지 게시판 내활동 기록 
		
9. 마이페이지 게시판 계정 관리
